Mode d'emploi — Facturation S/4HANA
❄ Principe : IVAN S4 génère le fichier CSV extract au format exact attendu par S/4HANA. Vous l'uploadez via la transaction FKKBIXBIT_upload.
  1. Récupérez le Business Partner (tapez l'ancien code client, IVAN le trouve) et le Contract Account
  2. Choisissez le type de facture
  3. Ajoutez vos lignes de charges (charge code + quantité + tarif)
  4. Renseignez les champs obligatoires (marqués *)
  5. Cliquez Générer le fichier (CSV)
  6. Dans S/4HANA : FKKBIXBIT_upload → Source → attachez le CSV → Read → Load
  7. Quand tout est vert, cliquez Billing Documents pour obtenir le numéro
