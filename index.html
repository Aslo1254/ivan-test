Mode d'emploi

Objectif : transformer une pile de PDFs Solar/IFS en factures FNE officielles créées automatiquement sur services.fne.dgi.gouv.ci, sans saisie manuelle.

1 Installer le bookmarklet (une seule fois)
  1. Affichez la barre de favoris : Ctrl + Maj + B
  2. Glissez le bouton ci-dessous directement dans votre barre de favoris
  3. C'est installé. À refaire seulement si vous changez de PC ou de navigateur.
↓ Glissez ce bouton dans vos favoris ↓
⚡ IVAN-FNE-BATCH
2 Charger les PDFs & lancer le batch
  1. Déposez vos PDFs Solar/IFS dans la zone ci-dessous.
  2. Vérifiez la file générée, puis ouvrez le portail DGI et connectez-vous.
  3. Cliquez sur le bookmarklet IVAN-FNE-BATCH sur la page DGI, puis lancez le traitement.

Mode d'emploi — Facturation S/4HANA
❄ Principe : IVAN S4 génère le fichier CSV extract au format exact attendu par S/4HANA. Vous l'uploadez via la transaction FKKBIXBIT_upload.
  1. Récupérez le Business Partner (tapez l'ancien code client, IVAN le trouve) et le Contract Account
  2. Choisissez le type de facture
  3. Ajoutez vos lignes de charges (charge code + quantité + tarif)
  4. Renseignez les champs obligatoires (marqués *)
  5. Cliquez Générer le fichier (CSV)
  6. Dans S/4HANA : FKKBIXBIT_upload → Source → attachez le CSV → Read → Load
  7. Quand tout est vert, cliquez Billing Documents pour obtenir le numéro
